# Installation

```
git clone https://github.com/Avaiga/demo-movie-genre.git
cd demo-movie-genre
pip install -r requirements.txt
```

If you want to run tests, please install `Pipenv`:
```
pip install pipenv
git clone https://github.com/Avaiga/demo-movie-genre.git
cd demo-movie-genre
pipenv install --dev
pipenv run pytest
```
